🚄 新幹線 領収書ダウンロード

指定期間の領収書をまとめてデスクトップに保存します（スマートEX / エクスプレス予約 / えきねっと）。

実行するとお使いのPCにブラウザが開きます。会員ID・パスワード（必要に応じSMS認証）を手入力してログインしてください。 ログイン後は自動で進み、From年月〜To年月の領収書をすべてデスクトップに保存します（単月は From と To を同じに）。
・スマートEX / エクスプレス予約（JR東海）: 宛名を入力して領収書を表示し PDF 化。23:30〜5:30 はメンテで利用不可。
・えきねっと（JR東日本）: Akamai 対策のため実 Google Chrome が起動します（要 Chrome）。宛名は1行目に入力。
※ ログイン情報は保存しません。
